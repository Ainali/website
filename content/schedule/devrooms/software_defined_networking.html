FOSDEM 2020 Software Defined Networking devroom

The FOSDEM SDN devroom hosts a diversity of talks covering open source networking projects and technologies. Software Defined Networking continues to be an exciting, growing and diverse field.

This year the devroom will focus on the following subject areas.

  • Cloud, Kubernetes & CNI
  • Telemetry
  • eBPF
  • QUIC
  • Software data-plane (DPDK, Linux Kernel, FD.io and friends)
