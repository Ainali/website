--- title: Pushing IPv6 ---

For the first time this year, the FOSDEM wireless network will be IPv6-only.

---MORE---

FOSDEM is a unique opportunity to confront thousands of developers with an IPv6-only reality. We are hopeful that making our default network IPv6-only will encourage people to fix bugs in applications and devices.

The FOSDEM network has NAT64 and DNS64 transition measures in place for communicating with the legacy internet. For those who need it, the FOSDEM-dualstack wireless network will provide the same dual stack connectivity we offered on FOSDEM in previous years.

  • FOSDEM: IPv6-only, provides NAT64 and DNS64. Recursive DNS resolvers announced with RA and DHCPv6. If you need to configure DNS manually, use 2001:67c:1810:f056::2 and 2001:41d0:2:1959:21c:c0ff:fe88:6f58
  • FOSDEM-v6: Identical to FOSDEM! Should FOSDEM need to revert to dual stack for any reason (hopefully not), this network will remain available as IPv6-only.
  • FOSDEM-dualstack: provides native IPv6 and legacy IP connectivity. No DNS64.
